# Быстрый запуск Frontend

## Запуск локального сервера

### Вариант 1: Python (рекомендуется)

```bash
cd web-client
python3 -m http.server 8000
```

Откройте: http://localhost:8000

### Вариант 2: Node.js

```bash
cd web-client
npx http-server -p 8000
```

### Вариант 3: PHP

```bash
cd web-client
php -S localhost:8000
```

## Важно!

Перед использованием фронтенда убедитесь, что:

1. **Backend запущен** на http://localhost:8083
   ```bash
   cd user-service
   ./gradlew bootRun --args='--spring.profiles.active=dev'
   ```

2. **CORS настроен** (если нужно)
   - По умолчанию Spring Boot разрешает все запросы

## Использование

1. Откройте http://localhost:8000
2. Зарегистрируйте нового пользователя
3. Войдите в систему
4. Просматривайте профиль, прогресс и достижения

## Структура страниц

- **index.html** - Регистрация и вход
- **profile.html** - Профиль пользователя (можно редактировать, включая аватарку)
- **progress.html** - Прогресс (XP, уровни, стрик, навыки)
- **achievements.html** - Достижения и лидерборд

## Особенности

- ✅ Современный дизайн с градиентами
- ✅ Адаптивная верстка (работает на мобильных)
- ✅ Визуализация прогресса с прогресс-барами
- ✅ Отображение навыков с уровнями
- ✅ Лидерборд топ-10 пользователей
- ✅ Работа с аватарками (через URL)
