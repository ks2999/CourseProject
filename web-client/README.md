# Frontend - Платформа обучения программированию

Минимальный фронтенд для платформы обучения программированию.

## Структура

```
web-client/
├── index.html          - Главная страница (авторизация/регистрация)
├── profile.html        - Профиль пользователя
├── progress.html       - Прогресс (XP, уровни, навыки)
├── achievements.html   - Достижения и лидерборд
├── css/
│   └── style.css      - Стили
└── js/
    ├── api.js          - Работа с API
    ├── auth.js         - Авторизация
    ├── profile.js      - Профиль
    ├── progress.js     - Прогресс
    └── achievements.js - Достижения
```

## Запуск

### Вариант 1: Простой HTTP сервер

```bash
cd web-client
python3 -m http.server 8000
```

Или с Node.js:
```bash
npx http-server -p 8000
```

### Вариант 2: Live Server (VS Code)

Установите расширение "Live Server" и откройте `index.html`

## Использование

1. Откройте http://localhost:8000
2. Зарегистрируйтесь или войдите
3. Просматривайте профиль, прогресс и достижения

## API Endpoints

Frontend использует следующие endpoints:

- `POST /api/users` - Регистрация
- `GET /api/users` - Получить всех пользователей (для входа)
- `GET /api/users/{id}` - Получить пользователя
- `PUT /api/users/{id}` - Обновить профиль
- `GET /api/progress/user/{userId}` - Прогресс пользователя
- `GET /api/progress/leaderboard` - Лидерборд
- `GET /api/student-skills/user/{userId}` - Навыки студента
