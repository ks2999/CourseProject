# Функциональность решения задач

## Описание

Создана полноценная система для решения задач по программированию на языке C.

## Возможности

### Страница задач (tasks.html)

1. **Список задач**
   - Отображение всех опубликованных задач
   - Фильтрация по сложности (Легкие, Средние, Сложные, Экспертные)
   - Показ награды в XP за каждую задачу

2. **Редактор кода**
   - Текстовый редактор для написания кода на C
   - Шаблон кода для каждой задачи
   - Кнопка сброса кода к шаблону

3. **Отправка решения**
   - Отправка кода на проверку
   - Отображение результатов тестов
   - Показ количества пройденных тестов
   - Сообщения об ошибках компиляции

4. **История решений**
   - Просмотр всех предыдущих попыток
   - Статус каждой попытки (ПРОЙДЕНО, НЕ ПРОЙДЕНО, ОШИБКА)
   - Дата и время отправки

### Страница уроков (lessons.html)

1. **Список уроков**
   - Отображение всех опубликованных уроков
   - Поиск по теме
   - Переход к задачам урока

2. **Просмотр урока**
   - Модальное окно с содержимым урока
   - Переход к задачам урока

## API Endpoints

### Задачи
- `GET /api/tasks/published` - Получить все опубликованные задачи
- `GET /api/tasks/{id}` - Получить задачу по ID
- `POST /api/submissions?userId={userId}` - Отправить решение

### Уроки
- `GET /api/lessons/published` - Получить все опубликованные уроки
- `GET /api/lessons/{id}` - Получить урок по ID
- `GET /api/lessons/topic/{topic}` - Получить уроки по теме

## Логика проверки кода

В текущей реализации используется симуляция проверки кода:

1. **Базовая валидация**
   - Проверка на пустой код
   - Проверка наличия основных элементов C кода

2. **Симуляция тестов**
   - 80% шанс успешного прохождения (для демонстрации)
   - Генерация случайного количества пройденных тестов

3. **Начисление XP**
   - XP начисляется только при первом успешном решении
   - Обновляется счетчик решенных задач
   - Обновляется прогресс пользователя

## Использование

1. Запустите backend:
   ```bash
   cd user-service
   ./gradlew bootRun --args='--spring.profiles.active=dev'
   ```

2. Запустите frontend:
   ```bash
   cd web-client
   python3 -m http.server 8000
   ```

3. Откройте http://localhost:8000 и войдите в систему

4. Перейдите на страницу "Задачи" и выберите задачу для решения

5. Напишите код в редакторе и нажмите "Отправить решение"

6. Просмотрите результаты тестов и историю решений
