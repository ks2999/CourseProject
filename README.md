# Курсовой проект
## Микросервисное приложение с элементами геймификации

---

##  Описание проекта
Данный курсовой проект представляет собой учебное микросервисное backend-приложение с элементами геймификации.  
Система предназначена для управления пользователями, аутентификации и повышения вовлечённости пользователей за счёт механик уровней, опыта и достижений.

Проект реализован с использованием микросервисной архитектуры, где каждый сервис отвечает за отдельную зону ответственности и взаимодействует с другими сервисами по REST API.

---

## Цели проекта
- Изучение принципов микросервисной архитектуры
- Практическое применение Spring Boot и Spring Security
- Реализация JWT-аутентификации
- Работа с базой данных через JPA / Hibernate
- Реализация элементов геймификации на уровне бизнес-логики
- Контейнеризация приложения с помощью Docker

---

##  Архитектура системы
Проект состоит из следующих компонентов:

- **API Gateway** — единая точка входа для всех запросов
- **Auth Service** — аутентификация и авторизация пользователей
- **User Service** — управление пользователями и их профилями
- **Gamification Service** — логика геймификации (опыт, уровни, достижения)
- **Web Client** — минимальный frontend для визуализации геймификации
- **PostgreSQL** — отдельная база данных для каждого сервиса

---

##  Функциональность (MVP)

### Аутентификация и авторизация
- Регистрация пользователей
- Вход в систему
- JWT-аутентификация
- Ролевая модель (USER / ADMIN)

### Управление пользователями
- Получение профиля пользователя
- Обновление данных пользователя
- Разделение прав доступа по ролям

###  Геймификация
- Начисление опыта (XP) за действия пользователя
- Система уровней
- Прогресс пользователя
- Достижения (achievements)

Примеры механик:
- Начисление XP при регистрации
- Повышение уровня при достижении определённого количества XP
- Получение достижений при выполнении условий

---

## Frontend
Frontend является минимальным и используется исключительно для:
- авторизации пользователя
- отображения профиля
- визуализации прогресса (XP, уровень)
- отображения достижений

Вся бизнес-логика геймификации реализована на backend-уровне.
